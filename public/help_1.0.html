HOLY QURAN ADVANCED SEARCH PROJECT

1. Corrections

As you start typing in text box, system will figure out if what you are typing is correct or not. In case it is not correct, system will suggest corrections. Example shown in the picture below

2. As-you-type completion

Also worth noting in the above picture is suggestions system gives you when you start typing a letter or few alphabets. Entering space on keyboard triggers search. On HTML datalist compatible browsers, first enter selects the suggestion and second enter triggers search.

3. Etymology

On the right side of the page, there is an option to view etymology of the word. For example, the following image shows etymology details of Arabic word فبای

4. Words, Stems, Roots

Ability to search by roots and stems along with other features are also implemented as shown in the image below. User can filter his/her search based on these parameters.

5. Significant Chapters

In search summaries tab on the right side of the page, significant chapters associated with search result are shown.

6. Translation words and their link

The significant words used in English, Urdu and German translations are specified below the significant chapters. The words are in form of hyperlink and by clicking on them, system will include them in search

7. Verse Details

By clicking on the button in red box in the image below, details of verse can be viewed. The details have the option to listen to recitation of the verse, English translation, Urdu translation and links to other useful related resources.

8. Advanced Search features

System has the ability to search in English, Urdu and German translations. Among language translations, we also have the option to search for exact or partials matches.

9. Chapter Search

Option to search among chapter names is also provided as shown in the image below.

10. Highlights

When searching for a word, the word being searched for is highlighted in verses in search results as shown in the image below.

11. Multi-lingual Search

System has ability to perform multi-lingual search. For example if you type one word in Arabic and another in English, the system will produce search results for Arabic text in Holy Quran with matching translation in same verse for English word. Image below explains that

12. Search Refinement Options

Search refinement options are also provided on right side of the page in search summaries tab.

13. Word Root

By clicking on the Arabic word, user can view roots of the word. For example in the image shown below, clicking on the highlighted word gave us its roots

14. Search Result Colors

Based on color of the highlighted word in result set, it will tell you whether the search was done based on stem, roots, partial or other. Notice the difference in color of highlighted word in image below.

15. Phonetic Search of Arabic words

Ability to search based on English phonetics has also been implemented in the system. For example, the word ‘nwr’ is similar in pronunciation to Urdu/Arabic word نور

16. Search History

Search history can also be seen on right side of the page in search history tab.

17. On Screen Keyboard

If someone does not have Urdu/Arabic/German keyboard, he/she can type Urdu/Arabic/German by clicking on keyboard icon in the search bar.

18. Chapter Name and Number Search

Ability to search chapter names and number is included in the system. For this user may have to uncheck all boxes other than “Chapter Names” one. Following images show ability to search Chapter Names by name and number
